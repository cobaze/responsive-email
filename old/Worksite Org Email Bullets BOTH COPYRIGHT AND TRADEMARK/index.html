DocAuto is pleased to introduce the latest release of WorkSite Organizer™ for Citrix ShareFile®. With a single click, you can upload, e-mail, or download documents between iManage Work and Citrix ShareFile. The WorkSite Organizer for ShareFile is a lightweight add-in to iManage DeskSite or FileSite.
The latest release of DocAuto’s WorkSite Organizer for Citrix ShareFile has been completely redesigned and rewritten using ShareFile’s newest application programming interface (API), and includes the following enhancements:
  • ShareFile mirrors the matter-centric structure of iManage by automatically creating folders and subfolders in ShareFile to match the content being uploaded.
  • iManage users can share content with secure links that require recipients to login. Authorized recipients can be selected from multiple sources, including Outlook contacts.
  • The user interface more closely matches the simple-to-use ShareFile web interface.
  • Very large files can now be uploaded, up to 10GB and higher on some accounts.
